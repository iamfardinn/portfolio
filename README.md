# Fahim Abrar - Personal Portfolio

A personal portfolio built to showcase my projects, philosophy, and experience as a Full-Stack Software Engineer. 

## Tech Stack
- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS v4**
- **GSAP / Framer Motion** (Animations)

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
